# SICAFIN Web — UI profesional

Versión mejorada del sitio estático público de SICAFIN.

## Qué se mejoró

- Nuevo diseño visual tipo fintech/profesional.
- Header sticky con navegación responsive.
- Hero principal con mockup de dashboard hecho en HTML/CSS.
- Cards de beneficios, módulos y planes más limpias.
- Página de contacto con formulario visual y canales claros.
- Página de privacidad con índice lateral.
- Footer más completo y consistente.
- Animaciones suaves con soporte para reducción de movimiento.

## Estructura

- `index.html`: portada principal.
- `quienes-somos.html`: historia, contexto y equipo.
- `productos.html`: planes, precios, módulos y CTA.
- `privacidad.html`: aviso de privacidad.
- `contacto.html`: contacto y solicitud de demostración.
- `styles/main.css`: estilos globales.
- `scripts/main.js`: navegación, header y animaciones.
- `assets/img/logo-horizontal.svg`: logo vectorial incluido.
- `assets/icons/favicon.ico`: favicon tomado del proyecto Desktop.
